Installation

Get running in under 2 minutes ‚Äî pick your mode

VS Code Extension

Prerequisites

  • VS Code version 1.90 or later
  • GitHub Copilot extension installed and active
  • Figma Personal Access Token ‚Äî get one here
  • .NET 10 SDK (for backend compilation & tests)
  • Node.js 20+ (for frontend compilation & tests)

Install & Run

Option A: VS Code UI

Open VS Code → Extensions panel → ... menu → Install from VSIX... → select novaforge-1.0.0.vsix

Option B: Command Line
code --install-extension novaforge-1.0.0.vsix

Then open the NovaForge icon in the Activity Bar, paste a Figma URL, and click Run Pipeline.

Download novaforge-1.0.0.vsix 13.2 MB

Web Agent Dashboard

Prerequisites

  • Node.js 20+ and npm
  • .NET 10 SDK
  • API key for OpenAI, Anthropic, or AWS Bedrock
  • Figma Personal Access Token
  • GitHub Token (optional, for PR creation)

Setup & Run

1. Configure environment
cp .env.example .env
# Add your API keys to .env
2. Install & start
npm install
npm run build
npm run start:all
3. Open dashboard
http://localhost:3000
