Installation

Get running in under 2 minutes

Prerequisites

  • VS Code version 1.90 or later
  • GitHub Copilot extension installed and active
  • Figma Personal Access Token ‚Äî get one here
  • .NET 10 SDK (for backend compilation & tests)
  • Node.js 20+ (for frontend compilation & tests)

Step 1 ‚Äî Install the Extension

Option A: VS Code UI

Open VS Code → Extensions panel → ... menu → Install from VSIX... → select novaforge-1.0.0.vsix

Option B: Command Line
code --install-extension novaforge-1.0.0.vsix

Step 2 ‚Äî Configure

  1. Open the NovaForge icon in the Activity Bar (left sidebar)
  2. Enter your Figma Personal Access Token in Settings (novaforge.figmaToken)
  3. Select a default model (GPT-4o is the free default)
  4. Optionally set per-agent models for specific agents

Step 3 ‚Äî Run!

  1. Paste a Figma design URL into the sidebar
  2. Choose an output folder
  3. Toggle any optional agents (Jira, Security, Code Review, Docs)
  4. Click Run Pipeline
  5. Watch the magic happen in real time!
