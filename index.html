Installation

Get running in under 2 minutes ‚Äî pick your mode

VS Code Extension

Prerequisites

  • VS Code version 1.90 or later
  • GitHub Copilot extension installed and active
  • Figma Personal Access Token ‚Äî get one here
  • .NET 10 SDK (for backend compilation & tests)
  • Node.js 20+ (for frontend compilation & tests)

Install & Run

Option A: VS Code UI

Open VS Code → Extensions panel → ... menu → Install from VSIX... → select novaforge-1.0.3.vsix

Option B: Command Line
code --install-extension novaforge-1.0.3.vsix
Configure Figma Token

Open VS Code Settings (Ctrl+,) → search novaforge → paste your Figma Personal Access Token into the Novaforge: Figma Token field.

Then open the NovaForge icon in the Activity Bar, paste a Figma URL, and click Run Pipeline.

Web Agent Dashboard

Prerequisites

  • Node.js 20+ and npm
  • .NET 10 SDK
  • API key for OpenAI, Anthropic, or AWS Bedrock

1. Configure environment
cp .env.example .env

2. Install & start

npm run build
npm run start:all

http://localhost:3000
